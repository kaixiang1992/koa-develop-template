# koa-develop-template
koa开发基本模板

```
# 依赖安装
npm install

# 项目启动
node app.js

# 页面访问
http://127.0.0.1:3000
```

* 项目结构
    * api 页面接口
    * config 环境变量配置信息等
    * controller 控制器
    * db 数据库配置连接，表结构设计
    * logs 记录日志
    * middleware 中间件
    * model 定义model数据，数据库增删改查
    * router 页面路由跳转
    * static 存放静态资源js、css、image等
    * views 存放视图页面
